Users make hand gestures representing rock, paper, or scissors, which the model recognizes. In this project I have created a file organising class which is designed to organize files in a specified directory by their extensions. The program begins by listing all file extensions in the directory and creates corresponding folders for each unique extension. It then sorts the files into these folders based on their extensions. The class demonstrates capabilities in file system navigation, directory creation, and file manipulation using Python's os and shutil libraries. This project is the Hangman game everybody knows and loves. It initializes with a list of words that the user can change as they wish, with a set number of lives. The class includes methods to check if a guessed letter is in the chosen word, update the word display, track the guessed letters, and reduce lives for incorrect guesses. In this project, I developed a Python class that calculates Body Mass Index (BMI) and categorizes the result into weight groups. Finally, the program evaluates the BMI and assigns it to a category: underweight, healthy range, overweight, obese, or severely obese. This project demonstrates my skills in user input handling, mathematical computation, and conditional logic.

SQL
Projects

Here you will find a repository of all the SQL projects I have completed. Some of these display basic SQL capabilities, while others demonstrate more advanced SQL skills.

This SQL data cleaning project works as a preface to the subsequent project and does an early stage refining a layoffs dataset through steps such as removing duplicates, standardizing data, eliminating NULLs, and dropping unnecessary columns. A staging table is used to maintain the integrity of the raw data while transformations are applied. Key techniques include using ROW_NUMBER() for deduplication and ensuring consistent data formats. The cleaned data is stored in the layoffs_staging table, ready for further analysis. This systematic approach ensures the dataset is accurate, consistent, and optimized for querying.

This SQL project performs exploratory data analysis on layoffs, using the set-up from the previous project. Here, I use techniques like CTEs, aggregate functions, and window functions to explore trends in the table, revealing key insights such as the maximum layoffs of 12,000, industry distribution, and geographic disparities. CTEs and functions like SUM, AVG, and ROW_NUMBER help analyze data by industry, company, and time. The project demonstrates SQL skills in data retrieval, aggregation, and transformation, providing insights into layoff trends and their implications.
